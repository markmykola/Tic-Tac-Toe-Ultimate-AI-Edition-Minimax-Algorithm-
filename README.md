# 🎮 Tic-Tac-Toe: Ultimate AI Edition (Minimax Algorithm)

Професійна імплементація класичної гри "Хрестики-нулики" з використанням бібліотеки `Pygame`. Головною особливістю проєкту є інтеграція **штучного інтелекту на базі алгоритму Minimax**, що робить його непереможним на максимальному рівні складності.

Проєкт демонструє навички побудови ігрової логіки, створення графічних інтерфейсів (GUI) та імплементації дерев прийняття рішень.

---

## 🚀 Ключові особливості (Key Features)

* 🧠 **Unbeatable AI (Minimax):** На складному рівні комп'ютер прораховує всі можливі варіанти розвитку подій до кінця гри за допомогою алгоритму Minimax. Виграти неможливо — найкращий можливий результат для гравця це нічия.
* 🎚️ **Багаторівнева складність:**
  * **Легкий:** AI робить випадкові ходи.
  * **Середній:** Ймовірнісна модель (60% — ідеальний хід, 40% — випадковий).
  * **Неможливий:** Чистий Minimax алгоритм.
* ⚙️ **State-Machine Architecture:** Надійна архітектура керування станами гри (`MENU`, `DIFFICULTY`, `RULES`, `PLAYING`), що гарантує відсутність багів при переходах між екранами.
* 🎨 **Сучасний UI/UX:**
  * Інтерактивні кнопки з ефектом наведення (hover).
  * Окрема нижня панель управління під час гри (рахунок, очищення поля, повернення в меню).
  * Динамічне перекреслення виграшної комбінації в кінці раунду.
